Heavenletters GraphQL API Test

Example Queries:

Get Latest Heavenletters:

{`query {
  heavenletters(limit: 3) {
    nid
    title
    number
    locale
    created
    author
    teaser
  }
}`}

Get Single Heavenletter:

{`query {
  heavenletter(nid: 1234) {
    nid
    title
    number
    body
    locale
    created
    author
  }
}`}

Get Heavenletter by Number & Language:

{`query {
  heavenletterByNumber(number: 4567, locale: "en") {
    nid
    title
    number
    locale
    author
  }
}`}

Get Translations for a Heavenletter:

{`query {
  heavenletterTranslations(nid: 890) {
    nid
    title
    locale
    language
  }
}`}
